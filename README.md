# 理发店管理系统

## 项目简介
一个面向理发店会员和消费记录管理的桌面化 Web 系统，当前版本保留源码并移除了业务数据。

## 原始功能
- 管理会员信息、服务项目、充值与消费记录。`r`n- 提供本地 Web 后端和前端页面。

## 二次开发功能
- 增加了桌面窗口壳、价格设置、导出能力和更完整的后台交互。`r`n- 预留了外部工具联动和本地部署逻辑。

## 运行方式
```powershell`r
python -m venv .venv`r
.\.venv\Scripts\activate`r
python -m pip install -r requirements.txt`r
python app.py`r
```

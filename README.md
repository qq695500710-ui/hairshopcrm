# 理发店管理系统

## 项目简介
一个面向理发店会员和消费记录管理的桌面化 Web 系统，当前版本保留源码并移除了业务数据。

## 可承接的相关业务
- 可有偿定制门店管理系统、会员系统、收银记录后台和本地桌面管理工具。
- 可承接理发店、美容店、足浴店、健身房等门店类管理系统开发。
- 适合做会员档案、消费记录、储值管理、报表导出和店铺私有化部署。

## 原始功能
- 管理会员信息、服务项目、充值与消费记录。
- 提供本地 Web 后端和前端页面。

## 二次开发功能
- 增加了桌面窗口壳、价格设置、导出能力和更完整的后台交互。
- 预留了外部工具联动和本地部署逻辑。

## 运行方式
```powershell
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install -r requirements.txt
python app.py
```
